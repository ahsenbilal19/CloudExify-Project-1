## Deployment

This project is deployed on **Vercel**.

Deployment steps:

1. Create a public GitHub repository named exactly `CloudExify-Project-1`.
2. Push the project files to the repository.
3. Open Vercel and click **Add New Project**.
4. Import the GitHub repository.
5. Select framework preset as **Other**.
6. Leave build command and output directory empty.
7. Click **Deploy**.
8. Copy the live `.vercel.app` link.
